Project Background

Smart Garage is an innovative project devloped to solve parking challenges using modern technologies like loT, sensors,automation, and real time-time monitoring.

Mission

To create smarter,safer and more efficient parking systems for homes, offices, and public places

Vision

To contribute to smart city development through inteligent parking solutions

Project Objectives

  • To design an automated parking system
  • To improve parking security
  • To reduce time spent searching for parking
  • To introduce smart technology into traditional garages

Technologies Used

  • Ardino
  • Ultrasonic Sensors
  • RFID Module
  • Servo motor for gate
  • CCTV Camera
  • Web application
  • Database for records
