# Smart Garage

**Slogan:** Park your car with no worries

Smart Garage is a simple Internet Technologies course work project built with HTML, CSS, JavaScript, and an optional PHP form handler.

## Important GitHub Pages Note

GitHub Pages hosts static websites only. The HTML, CSS, JavaScript, and SVG images will work on GitHub Pages, but `contact.php` will not run there.

For PHP support, host the project on a PHP server such as cPanel, XAMPP, WAMP, LAMP, or InfinityFree.

## How to Contribute Using GitHub Pull Requests

1. Fork the repository.
2. Create a new branch:

```bash
git checkout -b improve-homepage
```

3. Make your changes.
4. Commit your work:

```bash
git add .
git commit -m "Improve Smart Garage homepage"
```

5. Push your branch:

```bash
git push origin improve-homepage
```

6. Open a Pull Request on GitHub.
